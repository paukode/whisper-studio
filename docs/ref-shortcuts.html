Keyboard shortcuts

The small set of keys that speed up chat, recording, and the workspace. Whisper Studio keeps its shortcut list short on purpose, so the ones that exist are worth learning.

Global shortcuts use mod, which resolves to Cmd on macOS and Ctrl on Linux or Windows. So Cmd K and Ctrl K do the same thing depending on your machine. Because Whisper Studio is a macOS-first app, this page shows the Cmd form. A key only fires as a shortcut when nothing else has already consumed it, so typing inside an input, an open autocomplete, or a modal keeps its normal behavior.

Chat input

The chat textarea is where you will use keys the most. Sending is on Enter so a quick message needs no mouse, and newlines are still one modifier away.

KeysAction
EnterSend the message.
Shift EnterInsert a newline without sending.
Cmd EnterForce-send now, even while the slash-command autocomplete is open. An explicit "send this" that is never captured as a menu selection.
/ at line startOpen the slash-command menu. Keep typing to filter it. See Slash commands.

When the slash-command autocomplete is showing, Enter and Tab pick the highlighted command, Arrow Up / Arrow Down move the selection, and Esc closes the menu without sending. Composing with an IME (for example CJK input) never submits on the composition-confirm Enter.

The ESC kill switch

The single most useful key while an agent is working is Esc. When a turn is streaming, or any sub-agents are still running, pressing Esc (this is the same action as the Stop button) instantly kills the active stream and every running sub-agent for the session you are viewing.

KeysAction
Esc while streamingKill the current stream and all running sub-agents (same as pressing Stop).

The kill switch is armed only while something is actually running, so Esc keeps its ordinary meaning everywhere else. Overlays and popups that already treat Esc as "close me" always win: the first press dismisses the overlay, and a second press then stops the stream. For the full teardown path (how the stop signal reaches sub-agents and the sandbox), see Sub-agents & the event bus.

ESC stops, it does not undo. Any writes a tool already committed (files changed, a command that already ran) stay committed. The kill switch prevents further work, it does not roll back completed work. To limit what can run in the first place, use the Mode dial and approvals, described in Permissions & approvals.

Global app keys

These work from anywhere in the app (as long as focus is not captured by a text field or an open menu that needs the key). They open the interface pieces you reach for most.

KeysAction
Cmd KOpen the command palette.
Cmd BToggle the session sidebar.
Cmd ,Open Settings. See Settings & config keys.

On Linux or Windows, substitute Ctrl for Cmd in the rows above. The command palette (Cmd K) is the fastest way to reach an action whose exact key you do not remember, so it is worth making a habit of.

Voice submit (hands-free)

Dictation is not a key, but it is the closest thing to a hands-free "send". Click the microphone icon between Attach and Send to dictate into the chat textarea; the transcript appends as you speak. To submit without touching the keyboard, end a sentence with a spoken submit command like the examples below. The command words are stripped from the message before it is sent. A bare verb on its own (just "send") never fires, so a one-word mishear cannot submit by accident.

SayAction
"okay send", "send now", "submit now", "send it now", "send away", "fire away", "send the message"Submit the dictated message hands-free. A lead-in ("okay") or a closing word ("now", "away", "the message") is what turns a verb into a command; the command words are removed from the body.

The chat-input mic and the meeting Record button share one reference-counted audio stream, so you can dictate a message while a meeting is recording. See Voice & meetings for dictation, diarization, and system-audio capture.

These docs

This documentation site has its own small helper: press / anywhere (as long as you are not already typing in a field) to jump straight to the search box.

KeysAction
/Focus the docs search box.
Arrow Up / Arrow DownMove through search results.
EnterOpen the highlighted (or first) result.
EscClose the results and unfocus search.

Beyond keys

Whisper Studio deliberately keeps a small, memorable set of shortcuts rather than a large one. Most actions are reachable two other ways that need nothing to memorize:

Slash commands

Type / in the chat input for switching models, setting effort, exporting, clearing, and more. It is the fastest surface for chat-level actions. See Slash commands.

The toolbar and UI

Attach, mic, Send, Stop, the Mode dial, and the gear all have visible buttons. If you are unsure whether a key exists for something, the on-screen control always does.
